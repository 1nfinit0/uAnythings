\section{Introducción}

  Se presenta el proyecto de la asignatura de \textit{Bases de Datos II} basado en el desarrollo de un sistema de gestión de un negocio de venta de alimentos, en este caso, un restaurante. El sistema se encargará de gestionar la persistencia de los datos relacionados con cleintes, personal y productos, así como las operaciones de compra y venta. El objetivo principal es diseñar e implementar una base de datos eficiente y funcional mediante el uso del gestor de base de datos PostgreSQL, utilizando una interfaz web desarrollada con el framework Astro que permita a los usuarios interactuar con el sistema de manera intuitiva y eficiente. Este proyecto busca aplicar los conocimientos adquiridos en la asignatura para crear una solución práctica que pueda ser utilizada en un entorno real de negocio.
